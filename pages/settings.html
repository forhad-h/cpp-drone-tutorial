States Directory (git-based sync)

Save state files to the states/ folder in this repo and push to GitHub. Pull on another machine and load from here to resume exactly where you left off.

How it works
1
Download your state using the button above, rename it (e.g. home-pc.json), put it in the states/ folder.
2
Add the filename to states/manifest.json under the states array with a name and the filename.
3
Commit & push to GitHub. On another machine, pull and reload this page — your state will appear below.
4
Click Load next to the state you want.
manifest.json format 
{
  "states": [
    {
      "name": "Home PC — April 2026",
      "file": "home-pc-2026-04.json",
      "saved_at": "2026-04-15T10:00:00Z"
    },
    {
      "name": "Work Laptop — March 2026",
      "file": "work-laptop-2026-03.json",
      "saved_at": "2026-03-20T14:30:00Z"
    }
  ]
}

Located at states/manifest.json in the repo root.
